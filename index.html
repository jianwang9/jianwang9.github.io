Mysql笔记

发表于 2017-04-14 | 分类于 mysql
阅读时长

行转列

用if和case when then else end都行

select sum(CASE when score between 10 and 20 then 1 else 0 END) 一级别 from 表

select sum(if(score between 10 and 20,1,0)) 一级别 from 表

条件过滤

条件过滤啥大于,小于的,并不会匹配到包含null值的行

concat函数

用法: concat(string1,string2,…)
作用: 连接字符串,可以指定列的值和自己指定的字符串

now函数

用法: now()
作用: 获取当前系统时间

trim,rtrim,ltrim

用法: trim(String),ltrim(String),rtrim(String)
作用: 去除字符串两端,左边,右边的空白字符

curdate

用法: curdate()
作用: 获取当前日期,年月日

str_to_date,date_format

用法: str_to_date(string,format),date_format(date,format)
作用: 将指定的字符串转成日期类型,将日期类型转成字符串

year

用法: year(date)
作用: 取出日期类型中的year部门,其他内容截取也差不多

一些算术操作

abs() 返回一个数的绝对值
cos() 返回一个角度的余弦
exp() 返回一个数的指数值
pi() 返回圆周率
sin() 返回一个角度的正弦
sqrt() 返回一个数的平方根
tan() 返回一个角度的正切

一些聚合函数

avg() 返回某列的平均值
count() 返回某列的行数
max() 返回某列的最大值
min() 返回某列的最小值
sum() 返回某列值之和

insert into … select 和 selct xxx into 表a from 表b

前者导出 后者导入

存储过程

创建:
create procedure XXX(in xx 类型|out xx 类型)
begin
declare xx 类型;
sql语句;
end;

查看:
show procedure status;

删除:
drop procedure xxx;

索引

创建:
create index index_name on table (列)

查询:
show index from table;

删除:
dorp index index_name on table;

触发器

创建:
create trigger triggerName
after|before insert/update/delete on 表名
for each row
begin
sql语句;
end;

查询:
show triggers;

删除:
drop trigger trigger_name;
